To solve this problem, we need to follow these steps:

1. Generate the array `a` of length `n` where each element `a[i] = i * i - i + 1`.
2. Iterate through all possible triples `(a[i], a[j], a[k])` where `i < j < k`.
3. Check if the sum of each triple is a multiple of 3.
4. Count and return the number of valid triples.

Here's the Python function to achieve this:

```python
def get_max_triples(n):
    # Generate the array a
    a = [i * i - i + 1 for i in range(1, n + 1)]
    
    count = 0
    
    # Iterate through all possible triples (a[i], a[j], a[k]) where i < j < k
    for i in range(n):
        for j in range(i + 1, n):
            for k in range(j + 1, n):
                if (a[i] + a[j] + a[k]) % 3 == 0:
                    count += 1
    
    return count
```

This function first generates the array `a` using a list comprehension. It then uses three nested loops to iterate through all possible triples and checks if their sum is divisible by 3, incrementing the count for each valid triple. Finally, it returns the total count of valid triples.
