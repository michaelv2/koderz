Looking at this problem, I need to:

1. Check if all three inputs are integers
2. Check if any one number equals the sum of the other two
3. Return True only if both conditions are met

Let me trace through the examples:
- `any_int(5, 2, 7)`: All integers, and 7 = 5 + 2, so True
- `any_int(3, 2, 2)`: All integers, but no number equals sum of others, so False  
- `any_int(3, -2, 1)`: All integers, and 1 = 3 + (-2), so True
- `any_int(3.6, -2.2, 2)`: Not all integers, so False

```python
def any_int(x, y, z):
    # Check if all numbers are integers
    if not all(isinstance(num, int) for num in [x, y, z]):
        return False
    
    # Check if any number equals the sum of the other two
    return (x == y + z) or (y == x + z) or (z == x + y)
```
