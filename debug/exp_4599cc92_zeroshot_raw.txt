Let me reason through this problem:

1. I need to multiply two fractions given as strings
2. Check if the result is a whole number (integer)

For fractions a/b and c/d:
- (a/b) * (c/d) = (a*c)/(b*d)
- This is a whole number if (a*c) is divisible by (b*d)
- In other words, (a*c) % (b*d) == 0

My approach:
1. Parse both fraction strings to extract numerator and denominator
2. Multiply the numerators together and denominators together
3. Check if the resulting numerator is divisible by the resulting denominator

```python
def simplify(x, n):
    # Parse the first fraction
    x_parts = x.split('/')
    x_num = int(x_parts[0])
    x_den = int(x_parts[1])
    
    # Parse the second fraction
    n_parts = n.split('/')
    n_num = int(n_parts[0])
    n_den = int(n_parts[1])
    
    # Multiply the fractions: (x_num/x_den) * (n_num/n_den)
    result_num = x_num * n_num
    result_den = x_den * n_den
    
    # Check if the result is a whole number
    return result_num % result_den == 0
```
