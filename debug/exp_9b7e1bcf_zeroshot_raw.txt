Let me reason through this:

1. I need to return all prefixes of a string, starting from the shortest (first character) to the longest (full string)
2. For 'abc', the prefixes are: 'a', 'ab', 'abc'
3. I can iterate through positions 1 to len(string)+1 and slice the string accordingly

```python
def all_prefixes(string: str) -> List[str]:
    """ Return list of all prefixes from shortest to longest of the input string
    >>> all_prefixes('abc')
    ['a', 'ab', 'abc']
    """
    return [string[:i] for i in range(1, len(string) + 1)]
```
