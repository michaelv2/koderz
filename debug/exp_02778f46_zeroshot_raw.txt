Looking at this problem, I need to create a function that swaps the case of each character in a string. Lowercase characters should become uppercase, and uppercase characters should become lowercase. Non-alphabetic characters should remain unchanged.

Let me trace through the example:
- 'Hello' → 'hELLO'
- 'H' (uppercase) → 'h' (lowercase)
- 'e' (lowercase) → 'E' (uppercase)
- 'l' (lowercase) → 'L' (uppercase)
- 'l' (lowercase) → 'L' (uppercase)
- 'o' (lowercase) → 'O' (uppercase)

```python
def flip_case(string: str) -> str:
    """ For a given string, flip lowercase characters to uppercase and uppercase to lowercase.
    >>> flip_case('Hello')
    'hELLO'
    """
    result = ""
    for char in string:
        if char.islower():
            result += char.upper()
        elif char.isupper():
            result += char.lower()
        else:
            result += char
    return result
```
