Let me reason through this:

1. I need to iterate through each digit of the number
2. Check if each digit is odd
3. Multiply all odd digits together
4. Return 0 if no odd digits are found

Approach:
- Convert the number to a string to easily access individual digits
- For each digit, convert back to int and check if it's odd (digit % 2 == 1)
- Keep a running product of odd digits
- Initialize product to 1, then if no odd digits were found, return 0

```python
def digits(n):
    product = 1
    has_odd = False
    
    for digit_char in str(n):
        digit = int(digit_char)
        if digit % 2 == 1:  # digit is odd
            product *= digit
            has_odd = True
    
    return product if has_odd else 0
```
