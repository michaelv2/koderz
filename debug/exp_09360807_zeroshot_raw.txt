I will loop through the digits (using modulo and integer division), multiply the odd ones, and return 0 if no odd digits are found.

```python
def digits(n):
    """Given a positive integer n, return the product of the odd digits.
    Return 0 if all digits are even.
    """
    n = abs(int(n))
    prod = 1
    found = False
    if n == 0:
        return 0
    while n > 0:
        d = n % 10
        if d % 2 == 1:
            prod *= d
            found = True
        n //= 10
    return prod if found else 0
```
