To solve this problem, we need to follow these steps:

1. Check if the given sides form a valid triangle using the triangle inequality theorem.
2. If they do not form a valid triangle, return -1.
3. If they do form a valid triangle, calculate the area using Heron's formula.
4. Round the area to 2 decimal points and return it.

Here is the Python function implementation:

```python
import math

def triangle_area(a, b, c):
    # Check if the sides form a valid triangle
    if a + b <= c or a + c <= b or b + c <= a:
        return -1
    
    # Calculate the semi-perimeter
    s = (a + b + c) / 2
    
    # Calculate the area using Heron's formula
    area = math.sqrt(s * (s - a) * (s - b) * (s - c))
    
    # Round the area to 2 decimal points and return it
    return round(area, 2)
```
