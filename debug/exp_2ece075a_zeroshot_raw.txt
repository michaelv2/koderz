I will use an integer binary search to find whether there exists an integer whose cube equals the input. This avoids floating-point errors.

```python
def iscube(a):
    '''
    Write a function that takes an integer a and returns True 
    if this ingeger is a cube of some integer number.
    Note: you may assume the input is always valid.
    Examples:
    iscube(1) ==> True
    iscube(2) ==> False
    iscube(-1) ==> True
    iscube(64) ==> True
    iscube(0) ==> True
    iscube(180) ==> False
    '''
    if a >= 0:
        low, high = 0, a
    else:
        low, high = a, 0

    while low <= high:
        mid = (low + high) // 2
        cube = mid * mid * mid
        if cube == a:
            return True
        elif cube < a:
            low = mid + 1
        else:
            high = mid - 1
    return False
```
