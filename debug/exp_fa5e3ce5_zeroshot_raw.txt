Looking at this problem, I need to:
1. Take a list of elements
2. Remove duplicates (keep only unique elements)
3. Sort the unique elements in ascending order
4. Return the sorted list of unique elements

The function should work with any list of comparable elements (numbers in the example).

```python
def unique(l: list):
    """Return sorted unique elements in a list
    >>> unique([5, 3, 5, 2, 3, 3, 9, 0, 123])
    [0, 2, 3, 5, 9, 123]
    """
    return sorted(list(set(l)))
```
