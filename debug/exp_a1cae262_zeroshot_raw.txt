 The problem requires us to determine whether a given number can be expressed as the product of three prime numbers. To solve this, we will first create a helper function `is_prime` that checks if a number is prime or not. Then, we will iterate through all possible combinations of three prime numbers less than 100 and check if their product equals the given input number.

Here's the Python code for the solution:

```python
def is_multiply_prime(a):
    def is_prime(n):
        if n < 2:
            return False
        for i in range(2, int(n**0.5) + 1):
            if n % i == 0:
                return False
        return True

    primes = [i for i in range(2, 100) if is_prime(i)]
    for i in range(len(primes)):
        for j in range(i+1, len(primes)):
            for k in range(j+1, len(primes)):
                if primes[i] * primes[j] * primes[k] == a:
                    return True
    return False
```
