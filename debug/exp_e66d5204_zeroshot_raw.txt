Looking at this problem, I need to:
1. Remove duplicate elements from the list
2. Sort the remaining unique elements
3. Return the sorted list

Python's `set` data structure automatically removes duplicates, and the `sorted()` function will sort the result.

```python
def unique(l: list):
    """Return sorted unique elements in a list
    >>> unique([5, 3, 5, 2, 3, 3, 9, 0, 123])
    [0, 2, 3, 5, 9, 123]
    """
    return sorted(set(l))
```
