Let me break down this problem:

1. I need to calculate the "strength" of each extension as: (uppercase letters count) - (lowercase letters count)
2. Find the extension with the highest strength
3. If there's a tie, choose the first one in the list
4. Return the result in format: `ClassName.StrongestExtensionName`

Let me trace through the example:
- Class: "my_class"
- Extensions: ['AA', 'Be', 'CC']
  - 'AA': 2 uppercase, 0 lowercase → strength = 2 - 0 = 2
  - 'Be': 1 uppercase, 1 lowercase → strength = 1 - 1 = 0
  - 'CC': 2 uppercase, 0 lowercase → strength = 2 - 0 = 2

'AA' and 'CC' both have strength 2, so we pick 'AA' (comes first).
Result: 'my_class.AA' ✓

And the other example from the docstring:
- Class: "Slices"
- Extensions: ['SErviNGSliCes', 'Cheese', 'StuFfed']
  - 'SErviNGSliCes': S,E,N,G,S,C = 6 uppercase, r,v,i,l,i,e,s = 7 lowercase → 6 - 7 = -1
  - 'Cheese': C = 1 uppercase, h,e,e,s,e = 5 lowercase → 1 - 5 = -4
  - 'StuFfed': S,F = 2 uppercase, t,u,f,e,d = 5 lowercase → 2 - 5 = -3

'SErviNGSliCes' has strength -1 (highest), so result is 'Slices.SErviNGSliCes' ✓

```python
def Strongest_Extension(class_name, extensions):
    def strength(ext):
        uppercase = sum(1 for c in ext if c.isupper())
        lowercase = sum(1 for c in ext if c.islower())
        return uppercase - lowercase
    
    strongest = max(extensions, key=strength)
    return f"{class_name}.{strongest}"
```
