Looking at this problem, I need to:
1. Check if all numbers in list `l` are below threshold `t`
2. Return `True` if all numbers are below `t`, `False` otherwise

Let me trace through the examples:
- `below_threshold([1, 2, 4, 10], 100)` should return `True` because all numbers (1, 2, 4, 10) are below 100
- `below_threshold([1, 20, 4, 10], 5)` should return `False` because 20 is not below 5

I can solve this by using the `all()` function with a generator expression that checks if each element is less than the threshold.

```python
def below_threshold(l: list, t: int):
    """Return True if all numbers in the list l are below threshold t.
    >>> below_threshold([1, 2, 4, 10], 100)
    True
    >>> below_threshold([1, 20, 4, 10], 5)
    False
    """
    return all(x < t for x in l)
```
