I will implement an iterative Fibonacci function that returns the n-th Fibonacci number with fib(1) == 1. For n <= 0 it returns 0.

```python
def fib(n: int):
    """Return n-th Fibonacci number.
    >>> fib(10)
    55
    >>> fib(1)
    1
    >>> fib(8)
    21
    """
    if n <= 0:
        return 0
    a, b = 0, 1
    for _ in range(n):
        a, b = b, a + b
    return a
```
