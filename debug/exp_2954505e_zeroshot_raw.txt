To solve this problem, we need to follow these steps:

1. Check the sum of the first and last elements of the array.
2. If the sum is odd, sort the array in ascending order.
3. If the sum is even, sort the array in descending order.
4. Return a copy of the sorted array without modifying the original array.

Here's the implementation:

```python
def sort_array(array):
    if not array:
        return []
    
    # Determine the sorting order based on the sum of the first and last elements
    if (array[0] + array[-1]) % 2 == 0:
        sorted_array = sorted(array, reverse=True)
    else:
        sorted_array = sorted(array)
    
    return sorted_array
```

This function first checks if the array is empty. If it is, it returns an empty list. Otherwise, it calculates the sum of the first and last elements to determine the sorting order. It then creates a copy of the sorted array based on the determined order and returns it.
