Problem analysis:
- Core challenge: Determine sorting order based on the parity of the sum of the first and last elements of the input array, then return a new array sorted accordingly without mutating the original. Handle empty input gracefully.
- Constraints: Input is a list of non-negative integers. The original list must remain unchanged. Empty list should return an empty list.

Implementation specification:
- Function sort_array(array) returns a new list.
- If array is empty, return [].
- Compute s = array[0] + array[-1].
- If s is odd, return a new list sorted in ascending order.
- If s is even, return a new list sorted in descending order.
- Do not modify the input array.

```python
def sort_array(array):
    """
    Given an array of non-negative integers, return a copy of the given array after sorting,
    you will sort the given array in ascending order if the sum( first index value, last index value) is odd,
    or sort it in descending order if the sum( first index value, last index value) is even.

    Note:
    * don't change the given array.
    """
    if not array:
        return []
    s = array[0] + array[-1]
    if s % 2 == 1:
        return sorted(array)
    else:
        return sorted(array, reverse=True)
```
