Looking at this problem, I need to:

1. Create a mapping of GPA ranges to letter grades
2. For each GPA in the input list, find the appropriate letter grade
3. Return a list of letter grades

The key insight is that the grades are based on thresholds where:
- 4.0 exactly → A+
- > 3.7 → A
- > 3.3 → A-
- And so on...
- ≤ 0.0 → E

I'll iterate through each GPA and compare it against the thresholds in order, returning the first matching grade.

```python
def numerical_letter_grade(grades):
    letter_grades = []
    
    for gpa in grades:
        if gpa == 4.0:
            letter_grades.append('A+')
        elif gpa > 3.7:
            letter_grades.append('A')
        elif gpa > 3.3:
            letter_grades.append('A-')
        elif gpa > 3.0:
            letter_grades.append('B+')
        elif gpa > 2.7:
            letter_grades.append('B')
        elif gpa > 2.3:
            letter_grades.append('B-')
        elif gpa > 2.0:
            letter_grades.append('C+')
        elif gpa > 1.7:
            letter_grades.append('C')
        elif gpa > 1.3:
            letter_grades.append('C-')
        elif gpa > 1.0:
            letter_grades.append('D+')
        elif gpa > 0.7:
            letter_grades.append('D')
        elif gpa > 0.0:
            letter_grades.append('D-')
        else:
            letter_grades.append('E')
    
    return letter_grades
```
