I will check each word's length for primality (primes are integers >= 2 with no divisors other than 1 and themselves) and join the words whose lengths are prime in original order.

```python
def words_in_sentence(sentence):
    def is_prime(n):
        if n < 2:
            return False
        if n == 2:
            return True
        if n % 2 == 0:
            return False
        i = 3
        while i * i <= n:
            if n % i == 0:
                return False
            i += 2
        return True

    words = sentence.split()
    result = [w for w in words if is_prime(len(w))]
    return " ".join(result)
```
