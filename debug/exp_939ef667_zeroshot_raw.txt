Approach:
1. First, count the frequency of each element
2. Then, iterate through the original list and keep only elements that appear exactly once
3. This preserves the original order

```python
def remove_duplicates(numbers: List[int]) -> List[int]:
    """ From a list of integers, remove all elements that occur more than once.
    Keep order of elements left the same as in the input.
    >>> remove_duplicates([1, 2, 3, 2, 4])
    [1, 3, 4]
    """
    from collections import Counter
    
    count = Counter(numbers)
    return [num for num in numbers if count[num] == 1]
```
