```python
def sort_array(array):
    # Check if the array is empty or has only one element
    if len(array) < 2:
        return array[:]
    
    # Calculate the sum of first and last elements
    total = array[0] + array[-1]
    
    # Sort in ascending order if the sum is odd, otherwise sort in descending order
    sorted_array = sorted(array) if total % 2 != 0 else sorted(array, reverse=True)
    
    return sorted_array
```
This function first checks whether the array has less than two elements. If so, it returns a copy of the original array because there's nothing to sort in this case. Then it calculates the sum of the first and last elements of the array. If this sum is odd, it sorts the array in ascending order; otherwise, it sorts the array in descending order. Finally, it returns the sorted array.
