 The problem is about finding the longest string from a list of strings. If there are multiple strings with the same length, we should return the first one. If the input list is empty, we should return None.

Here's my plan:
1. Check if the input list is empty. If it is, return None.
2. Initialize two variables: `longest_string` to store the longest string and `max_length` to store its length. Set both of them to the first element of the input list.
3. Iterate over the rest of the elements in the input list. For each element, check if its length is greater than `max_length`. If it is, update `longest_string` and `max_length`.
4. Return `longest_string`.

Now I'll write the code to solve this problem.
```python
from typing import List, Optional

def longest(strings: List[str]) -> Optional[str]:
    if not strings:
        return None
    longest_string = strings[0]
    max_length = len(longest_string)
    for string in strings[1:]:
        if len(string) > max_length:
            longest_string = string
            max_length = len(string)
    return longest_string
```
