 The problem requires us to find the maximum negative integer and minimum positive integer from a given list. If there are no negative or positive integers, we should return `None`. Here's how I plan to solve this:

1. Initialize two variables, `max_neg` and `min_pos`, with `None`. These will store the maximum negative integer and minimum positive integer respectively.
2. Iterate over each number in the list.
3. If the number is negative and either `max_neg` is `None` or the number is greater than `max_neg`, update `max_neg`.
4. If the number is positive and either `min_pos` is `None` or the number is smaller than `min_pos`, update `min_pos`.
5. Finally, return a tuple of `(max_neg, min_pos)`.

Now let's implement this in Python:

```python
def largest_smallest_integers(lst):
    max_neg = None
    min_pos = None
    for num in lst:
        if num < 0:
            if max_neg is None or num > max_neg:
                max_neg = num
        elif num > 0:
            if min_pos is None or num < min_pos:
                min_pos = num
    return (max_neg, min_pos)
```
