```python
def is_prime(n):
    """Check if n is a prime number."""
    if n <= 1 or (n % 2 == 0 and n > 2): 
        return False
    for i in range(3, int(n**0.5) + 1, 2):
        if n % i == 0:
            return False
    return True

def fibonacci():
    """Generate Fibonacci numbers."""
    a, b = 0, 1
    while True:
        yield a
        a, b = b, a + b

def prime_fib(n):
    count = 0
    for fib in fibonacci():
        if is_prime(fib) and fib > 1:
            count += 1
            if count == n:
                return fib
```
This solution works by first defining helper functions `is_prime`, which checks whether a number is prime, and `fibonacci`, which generates Fibonacci numbers. The main function `prime_fib` then iterates over the Fibonacci sequence, checking each number to see if it's both a Fibonacci number (i.e., generated by our `fibonacci` generator) and prime (as determined by our `is_prime` function). When it finds an appropriate number, it increments a counter. Once the counter reaches `n`, it returns that number.
