I will iterate downward from n//2 to 1 and return the first divisor found. If n <= 1 (no proper positive divisor), return None.

```python
def largest_divisor(n: int) -> int:
    """ For a given number n, find the largest number that divides n evenly, smaller than n """
    if n <= 1:
        return None
    for i in range(n // 2, 0, -1):
        if n % i == 0:
            return i
```
